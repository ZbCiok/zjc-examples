### Running : 
#### Run a MongoDB database (Docker): 
```shell script
docker run -ti --rm -p 27017:27017 mongo:4.4
```
#### Start Quarkus: 
```shell script
quarkus dev
```
#### Open a browser to http://localhost:8080/fruits.html
#### Open a browser to http://localhost:8080/reactive_fruits
